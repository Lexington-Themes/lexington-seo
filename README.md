# @michael-andreuzza/seo

SEO component for Astro projects.

## Install

```bash
npm i @michael-andreuzza/seo
```

## Usage

```astro
---
import { AstroSeo } from "@michael-andreuzza/seo";
---

<AstroSeo
  title="My page title"
  description="My page description"
  canonical="https://example.com/page"
/>
```

## Development

```bash
npm install
npm test
```
